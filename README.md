# MusicFlow - 音乐律动沉浸式体验

一款基于 Three.js + LLM 的音乐可视化与歌词渲染工具。

## 功能特性

- 🎵 支持导入 MP3/MP4 音频文件
- 🤖 LLM 智能分析音乐情感与节奏，推荐可视化风格
- ✨ 多样化的 Three.js 背景粒子效果
- 🎤 LRC 歌词解析与精美渲染
- 🎬 视频导出功能

## 快速开始

```bash
npm install
npm run dev
```

## 使用说明

1. 导入音频文件 (MP3/MP4)
2. 导入歌词文件 (LRC，可选)
3. 配置 LLM API Key
4. 点击"分析音乐"使用 LLM 分析
5. 点击播放开始体验
6. 点击"导出视频"保存为 WebM 格式
